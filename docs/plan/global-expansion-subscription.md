# 기질아이 글로벌 확장 + 구독제 PRD

Status: Proposal
Date: 2026-03-25

## Problem

기질아이는 현재 한국어 전용 웹앱으로, 건별 결제(990원)만 지원한다. 반복 수익 구조가 없어 사업 지속성이 낮고, 영어권이라는 50배 큰 시장에 접근하지 못하고 있다. CBQ/ATQ 기질 검사는 원래 영어권 학술 도구이고, 영어권에 temperament + AI 조합 서비스가 사실상 없어 명확한 시장 빈틈이 존재한다.

## Current State

### 기존 정책 (docs/policies/)
- **결제**: Stripe PaymentIntent(카드) + 포트원 V1(카카오/토스/네이버페이, 가맹점 ID 플레이스홀더). 건별 990원만
- **무료/유료**: 리포트는 전체 동일. 건별 990원은 리포트+상담3회 패키지. 구독은 전부 무제한
- **쿨다운**: 무료 7일, 유료 24시간
- **상담**: 횟수 제한 없음 (무료 무제한)

### 관련 코드
- `app/src/app/payment/page.tsx` — Stripe Elements + 포트원 V1 SDK 혼재, 결제수단 선택 UI
- `app/src/app/api/payment/create-intent/route.ts` — Stripe PaymentIntent 생성
- `app/src/components/payment/CheckoutForm.tsx` — Stripe React 결제 폼
- `app/src/store/useAppStore.ts` — `isPaid: boolean` 플래그로 결제 상태 관리
- `app/src/app/share/page.tsx` — 카카오톡 공유 + html-to-image 카드 생성 (한국어 전용)
- `app/src/app/layout.tsx` — `lang="ko"` 하드코딩, 카카오 SDK만 로드
- `app/src/lib/prompts.ts` — AI 프롬프트 한국어 전용
- `app/src/data/questions.ts` — 설문 문항 한국어 전용

## Proposed Solution

3개 페이즈로 나누어 진행한다.

**Phase 1 — 결제 전환 + 구독제 (한국)**: 포트원 V2로 결제 통합, 구독 모델 도입. 기존 한국 사용자 대상으로 먼저 검증.

**Phase 2 — i18n + 글로벌 결제**: UI/설문/프롬프트 영문화, locale 기반 가격·통화·결제수단 분기. 웹으로 글로벌 소프트 런칭.

**Phase 3 — 글로벌 앱 배포 + 바이럴**: 앱스토어/플레이스토어 글로벌 출시, 영문 공유 카드, ASO 최적화. 앱을 주력 유입 채널로 활용.

## User Scenarios

### 한국 사용자 — 건별 구매
1. 부모가 설문 완료 → 기질 리포트 확인 → "990원으로 AI 상담 3회 열기" 클릭
2. 카카오페이로 990원 결제 → 즉시 프리미엄 리포트 열람
3. 다음 달 상담 3회째 시도 → "월 2회 무료 상담을 모두 사용했어요. 구독하면 무제한!" 안내

### 한국 사용자 — 구독 전환
4. 무료 상담 한도 도달 → 요금제 페이지에서 월 ₩9,900 구독 선택
5. 토스페이로 빌링키 발급 + 첫 결제 → 구독 활성화
6. 이후 리포트 무제한, 상담 무제한, 쿨다운 없음
7. 설정 > 구독 관리에서 해지 → 이번 결제 기간 끝까지 유지 후 만료

### 글로벌 사용자 — 구독
8. 미국 부모가 Instagram에서 "My child is a Curious Explorer" 카드 발견 → 링크 클릭
9. 영문 랜딩 → 설문 완료 → 기질 리포트 → "Unlock unlimited consultations - $9.99/mo"
10. Stripe(Google Pay)로 구독 → 영문 프리미엄 리포트 + 영문 AI 상담

### 바이럴 공유
11. 부모가 리포트 하단 "Share" 버튼 → 인스타 스토리용 카드 이미지 생성
12. 기질 유형 + 차원별 그래프가 담긴 영문 카드를 인스타 스토리에 공유
13. 친구가 "이거 뭐야?" → 링크 클릭 → 자연 유입

### Edge Case
- 구독 갱신 실패 → 3일간 재시도 → 실패 시 만료, 무료 티어로 전환
- 앱에서 접속한 글로벌 사용자 → IAP 강제 (Phase 3 별도 처리)
- 구독자가 건별 결제 페이지 접근 → 이미 구독 중이라는 안내 표시

## Scope

### Phase 1: 결제 전환 + 구독제 (한국)
- 포트원 V2 SDK 통합 (Stripe 직접 연동 + 포트원 V1 제거)
- 건별 결제 (₩990) 포트원 V2 전환
- 월/연 구독 도입 (₩9,900/월, ₩79,000/년)
- 구독 라이프사이클 (생성, 갱신, 해지, 만료)
- 기능 접근 제어 (리포트, 상담 횟수, 실천, 쿨다운)
- 요금제 선택 페이지 (`/pricing`)
- 구독 관리 페이지 (`/settings/subscription`)
- 웹훅 기반 결제 검증
- DB 스키마 (subscriptions, payments)
- **상세 스펙**: `docs/spec/payment-portone-v2.md` (작성 완료)

### Phase 2: i18n + 글로벌 결제
- locale 감지 (브라우저 언어 → cookie 저장)
- UI 전체 다국어 (한국어/영어)
- 설문 123문항 영문화 (CBQ/ATQ 원본 활용)
- AI 프롬프트 영문 버전 (리포트 3종 + 상담 3종)
- 기질 유형 영문 명칭/설명
- 정원 메타포 영문화 (식물/토양 명칭)
- 글로벌 가격 분기 ($4.99, $9.99/mo, $79.99/yr)
- Stripe 채널 활성화 (포트원 V2 경유)
- 글로벌 건별 결제 숨김 (수수료 비경제적)
- 글로벌 OAuth (Google 유지, Kakao → Apple ID 추가)

### Phase 3: 글로벌 앱 배포 + 바이럴
- Flutter 쉘 글로벌 앱스토어/플레이스토어 출시
- ASO 최적화 (키워드: temperament test, parenting guide, child personality)
- 영문 공유 카드 이미지 (인스타 스토리 최적화)
- 공유 채널 분기 (한국: 카카오톡 / 해외: Instagram, iMessage, WhatsApp)
- 웹 랜딩 페이지 → 앱 다운로드 유도
- 앱 내 IAP 연동 (Apple/Google 수수료 감안한 별도 가격 정책)

### Out of Scope
- **앱 네이티브 전환** — Flutter WebView 쉘 유지. 네이티브 리빌드는 트래픽 검증 후 판단
- **다국어 확장 (일본어, 중국어 등)** — 영어 검증 후 판단
- **B2B (어린이집/유치원 단체 라이센스)** — 잠재적 피벗 옵션이나 현재 범위 밖
- **자동 환불** — 초기에는 포트원 관리자 콘솔 수동 처리
- **프로모션/쿠폰 리뉴얼** — 기존 시스템 유지, 구독 쿠폰은 향후

### 정책 변경

| 정책 | 현재 → 변경 |
|------|------------|
| 결제 방식 | Stripe + 포트원 V1 → 포트원 V2 통합 |

### Risks

- **포트원 V2 안정성** — V2가 비교적 신규. SDK 문서/커뮤니티가 V1보다 적음
- **구독 전환율 불확실** — 무료→구독 2% 가정이나, 실제로는 더 낮을 수 있음
- **영문 프롬프트 품질** — 한국어 프롬프트의 섬세한 톤을 영어로 재현하기 어려울 수 있음
- **앱스토어 리젝** — WebView 앱은 리젝 리스크가 있음 (충분한 네이티브 기능 부족 사유)
- **IAP 수수료 충격** — 앱 내 구독 $9.99 중 최대 $3 손실. 가격 조정 필요할 수 있음
- **설문 이탈률** — 123문항은 영어권 사용자에게도 높은 허들. 축약 버전 검토 필요
- **무료 상담 제한 반발** — 기존 사용자가 갑자기 월 2회 제한을 받으면 이탈 가능. 기존 사용자 그랜드파더링 검토

## Success Criteria

- [ ] Phase 1: 포트원 V2 건별 결제가 한국 결제수단(카드/카카오/토스/네이버)으로 정상 작동
- [ ] Phase 1: 구독 생성 → 갱신 → 해지 → 만료 전체 라이프사이클 정상 작동
- [ ] Phase 1: 구독자는 리포트/상담 무제한, 비구독자는 제한 적용 확인
- [ ] Phase 2: 영어 UI로 설문 → 리포트 → 상담 전체 플로우 완료 가능
- [ ] Phase 2: 글로벌 사용자가 Stripe(USD)로 구독 결제 가능
- [ ] Phase 3: 앱스토어/플레이스토어 글로벌 출시 승인
- [ ] Phase 3: 영문 공유 카드가 Instagram 스토리에 정상 렌더링
- [ ] 전체: 구독 전환율 1% 이상 (MAU 대비 유료 구독자)

## Decisions

1. **유료 정책 전환 시점**: 무료→유료 제한(상담 월 2회 등) 적용 시점과 기존 사용자 처리는 별도로 결정 예정
2. **국내/글로벌 정책 통일**: 기본 정책(가격 구조, 기능 매트릭스)은 동일. 나라별 특성에 따라 유리한 부분만 변경 가능 (예: 결제수단, 앱스토어 가격)
3. **앱스토어 가격**: IAP 수수료 감안 인상 검토 (₩15,000 / $14.99 등). 구체적 금액은 Phase 3 진입 시 결정
4. **연 구독 환불 정책**: 해지 시 잔여 기간은 서비스 유지, 환불 없음. 결제일 기준 7일 이내 전액 환불 가능 (쿨링오프). 7일 이후 부분 환불 불가. 앱스토어 IAP는 각 스토어 환불 정책을 따름
5. **Phase 실행**: 병렬 진행 가능. Phase 1(결제)과 Phase 2(i18n) 동시 작업

## Open Questions

1. **설문 축약 버전**: 글로벌 출시 시 123문항 그대로 갈지, 단축 버전(예: 50문항)을 만들지? 학술적 타당성 vs 사용자 이탈률 트레이드오프
2. **글로벌 OAuth**: Apple ID 로그인 추가 필요 여부 및 우선순위
3. **글로벌 랜딩 페이지**: 기존 한국어 랜딩을 번역할지, 글로벌 전용 랜딩을 별도 제작할지

## 배포 전략

```
[웹사이트] ← 랜딩 + 바이럴 퀴즈 + SEO
    ↓ "앱에서 더 편하게"
[앱스토어/플레이스토어] ← 주력 유입 (ASO)
    ↓
[Flutter WebView 앱]
    ↓
[Next.js 웹앱] ← 모든 비즈니스 로직

결제 분기:
  웹 직접 방문 → 포트원 (수수료 3~7%)
  앱 내 → IAP (수수료 15~30%)
```

앱이 주력 유입 채널, 웹은 바이럴 + SEO 보조. 결제는 가능하면 웹으로 유도하여 수수료 절감.
