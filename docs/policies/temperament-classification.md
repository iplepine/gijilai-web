# 점수 산출 및 분류

## 1. 개별 문항 채점 (Raw Score)

- 모든 문항은 5단계 BARS(행동 고정 척도): 1점~5점
- 역채점 문항: `6 - 원래점수` (1↔5, 2↔4 뒤집기)

## 2. 척도별 합산 및 정규화

- 각 척도(NS, HA, RD, P)는 5문항씩
- 정규화 공식: `(총점 / (문항수 * 5)) * 100` → 0-100 스케일

### 4개 차원

| 차원 | 코드 | 설명 |
|------|------|------|
| 자극추구 | NS | 새로운 자극에 대한 접근성, 활동 수준, 충동성 |
| 위험회피 | HA | 불안, 수줍음, 감각 민감성, 변화 적응성 |
| 사회적 민감성 | RD | 정서적 감수성, 관계 지향성, 타인 의존성 |
| 인내력 | P | 과제 지속성, 주의 집중력, 좌절 내성, 완벽주의 |

## 3. High/Low 판정 기준

한국인 데이터 보정 기준(방법 B) 적용. 평균 점수를 0-100 스케일로 환산:

| 척도 | 평균 기준 | 0-100 환산 | 판정 |
|------|----------|-----------|------|
| NS | > 3.2 | > 64 | High |
| HA | > 2.8 | > 56 | High |
| RD | > 3.0 | > 60 | High |

이미지 파일 위치: `app/public/child_type/`

## 5. 양육자 기질 유형

ParentClassifier에서 NS-HA 조합 + RD/P 보정으로 결정. 아동 유형과는 별도 분류 체계:

| NS-HA 조합 | 유형 |
|-----------|------|
| NS↑ HA↓ | 에너지가 샘솟는 열정적인 마음 |
| NS↓ HA↑ | 평온을 지키는 섬세한 마음 |
| NS↑ HA↑ | 호기심과 신중함이 공존하는 마음 |
| NS↓ HA↓ | 흔들림 없는 평온하고 단단한 마음 |
| 기본값 | 따뜻하고 넉넉한 마음 |

보정: RD >= 65 → 공감형 보정, RD <= 35 → 논리형 보정, P >= 65 → 지속형 보정

## 6. 조화 분석 (Harmony)

부모-자녀 점수 차이가 가장 큰 차원을 핵심 관계 포인트로 식별:

| 차원 | 라벨 |
|------|------|
| NS | 에너지의 속도 차이 |
| HA | 안전 거리 감각 |
| RD | 마음의 온도차 |
| P | 성취의 호흡 |
