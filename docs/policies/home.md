# 홈 화면 정책

## 화면 분기

홈 화면은 사용자 상태에 따라 세 가지로 분기된다.

| 상태 | 조건 | 표시 |
|------|------|------|
| 비로그인 웹 | `user` 없음 + 일반 브라우저 | `LandingPage` 컴포넌트 렌더링 |
| 비로그인 앱 | `user` 없음 + `gijilai_app` WebView | 소개 화면 대신 `/login`으로 이동 |
| 신규 사용자 | 등록된 아이 없음 (DB + 로컬 intake 모두 없음) | 환영 히어로 카드 + 온보딩 모달 |
| 기존 사용자 | 아이 1명 이상 등록 | 아이 프로필 + 기능 카드 리스트 |

## 신규 사용자 레이아웃

1. **환영 히어로 카드** — 양육자 이름 포함 인사, 아이 프로필 등록 CTA (`/settings/child/new`)
2. **온보딩 모달** — 첫 방문 시 자동 표시. 모바일 WebView에서 화면보다 과하게 길어지지 않도록 컴팩트 간격과 최대 높이를 적용하며, "아이 정보 등록하기" 또는 "나중에 둘러볼게요" 선택 가능.

## 기존 사용자 레이아웃

### 상단 영역

- **아이 프로필 사진**: 탭하면 이미지 업로드. 사진 없으면 기질 유형 이미지 또는 기본 아이콘 표시.
- **아이 기질 뱃지**: 프로필 사진 아래 뱃지. 검사 완료 시 기질 유형 라벨, 미완료 시 "기질 검사하기" 텍스트. 탭하면 `/report?tab=child` (완료) 또는 `/survey/intro` (미완료).
- **아이 이름 + 나이**: `{이름} ({나이})` 형태. 나이는 36개월 이하면 개월 수, 초과하면 년+개월.
- **아이 선택 드롭다운**: 아이 2명 이상일 때 이름 옆에 `expand_more` 아이콘 표시. 탭하면 드롭다운으로 아이 전환. 선택된 아이는 Zustand `selectedChildId`로 전역 유지.
- **양육자 유형 뱃지**: 양육자 기질은 아이와 무관하게 하나만 표시. 데이터 없으면 "검사 필요" 표시. 양육자 데이터가 있고 아이 검사 미완료 시 탭하면 검사 안내(`/survey/intro`)로 이동, 완료 시 `/report?tab=parent`로 이동.

### 구독/체험 상태 뱃지

헤더 "기질아이" 로고 오른쪽 끝에 컴팩트 뱃지로 표시한다.

| 사용자 상태 | 뱃지 표시 | 탭 시 이동 |
|------------|----------|-----------|
| 체험 기간 중 | 체험중 D-{남은일수} | `/pricing` |
| 체험 만료 (비구독) | 프리미엄 시작하기 | `/pricing` |
| 구독자 (ACTIVE) | 구독중 | `/settings/subscription` |
| 구독자 (CANCELLED) | 구독중 ~MM/DD | `/settings/subscription` |

### 체험 종료 임박 구독 유도

- 체험 기간 중이고 남은 기간이 2일 이하이며 비구독자인 경우, 기능 카드 리스트 상단에 프리미엄 지속 카드 노출
- CTA는 `/pricing`으로 이동
- 목적은 체험 종료 후 상담/실천 기록/리포트 기반 가이드가 끊기지 않는다는 점을 알리는 것

### 기능 카드 리스트

카드는 아래 순서대로 배치하며, 조건에 따라 표시/숨김 처리한다.

| 순서 | 카드 | 표시 조건 | 비고 |
|------|------|-----------|------|
| 1 | {아이이름}의 기질 검사 | 선택된 아이의 아동 검사 미완료 | "첫 번째 단계" 뱃지, 검사 시작하기 CTA → `/survey/intro` |
| 2 | 나의 양육 성향 검사 | 양육자 검사 미완료 AND 아동 검사 완료 | "필수 진행 단계" 뱃지, 이어하기/시작하기 CTA → `/survey?type=PARENT` |
| 3 | 오늘의 마법의 한마디 | 실천 중인 상담에 magicWord 존재 | 캐러셀(5초 자동 롤링, 스와이프 지원), 선택된 아이로 필터링 |
| 4 | 오늘의 실천 | 오늘 미체크 실천 항목 있음 | `/practices`로 이동, 최대 3개 미리보기 표시 |
| 5 | 상담 유도 | 아동 검사 완료 AND (상담 이력 없음 OR 진행 중 실천 없음) | "육아 고민, 기질로 풀어볼까요?" → `/consult` |
| 6 | 기질 분석 리포트 | 아동 검사 완료 | 컴팩트 카드, "결과 보기" → `/report` |

> **참고**: 재검사 쿨다운 상태는 내부적으로 추적하고 있으나(`cooldownStatus`), 현재 홈 화면에 재검사 카드 UI는 렌더링하지 않는다.
