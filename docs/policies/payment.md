# 결제 및 가격 정책

## 결제 인프라

- 결제 방식: 포트원 V2 SDK
- 한국 PG: 토스페이먼츠 (카드, 카카오페이, 토스페이, 네이버페이)
- 글로벌 PG: Stripe (카드, Google Pay, Apple Pay)
- locale(ko/en)에 따라 PG사/통화/결제수단 자동 분기

## 가격

| 상품 | 한국 (KRW) | 글로벌 (USD) |
|------|-----------|-------------|
| 프리미엄 리포트 1회 | 1,980원 | 건별 미제공 (구독 유도) |
| 월 구독 | 12,000원/월 | $11.99/month |
| 연 구독 | 89,000원/년 (월 환산 ~7,417원, 38% 할인) | $89.99/year |

- 건별 1,980원 결제는 국내 결제수단(토스페이먼츠)으로만 허용
- Stripe 경유 소액 결제는 고정 수수료($0.30)로 인해 비경제적이므로 글로벌 미노출

## 7일 리버스 트라이얼

- 신규 가입 시 7일간 전체 기능(프리미엄 동일) 무료 체험
- 카드 정보 사전 요구 안 함
- 체험 기간 판단: `user.created_at` 기준 7일 이내
- 체험 종료 후 자동으로 베이직(무료) 전환
- 홈 헤더 뱃지: "체험중 D-N" 표시, 탭 시 `/pricing` 이동
- 체험 종료 2일 전 안내 표시 (상담 시작 버튼 위)

## 티어별 기능

| 기능 | 베이직 (무료) | 건별 구매 (₩1,980) | 구독 |
|------|-------------|-----------------|------|
| 기질 리포트 | O (전체 동일) | O | O |
| AI 상담 | 체험 기간 무제한, 이후 이용 불가 | +1회 (리포트별) | 무제한 |
| 실천 기록 | 최근 1개 | 최근 1개 | 전체 이력 |
| 재검사 쿨다운 | 7일 | 24시간 | 24시간 |

## 구독 라이프사이클

- 구독 생성: 빌링키 발급 → 첫 결제 → ACTIVE
- 정기 갱신: 매일 cron으로 만료 구독 자동 결제
- 갱신 실패: PAST_DUE 상태, 3일간 재시도 (총 3회). 3회 실패 시 EXPIRED
- 해지: 즉시 해지가 아닌 기간 만료 해지 (cancelled_at 방식). 남은 기간 동안 서비스 유지
- 환불: 결제 7일 이내 전액 환불 가능 (쿨링오프). 이후 환불 불가

## 쿨다운 및 재검사

- 무료 사용자: 동일 아이 기준 7일 쿨다운
- 건별 구매자: 프리미엄 리포트 생성 후 24시간 쿨다운
- 구독자: 쿨다운 없음

## AI 리포트 생성

- 모델: gpt-4o-mini (전 티어 통일)
- Temperature: 0.7
- 응답 형식: JSON 객체 (response_format으로 강제)
- 3가지 프롬프트: CHILD, PARENT, HARMONY
- JSON 파싱 실패 시 원본 문자열로 폴백
- 모든 리포트 API 호출은 Supabase 세션 인증 필요
