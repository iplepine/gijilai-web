# 결과 리포트 정책

결과 리포트 페이지(`/report`)는 3개 탭으로 구성되며, 각 탭별 섹션의 규칙을 정의한다.

---

## 공통 정책

### 용어 표기
- 영문 약어(NS, HA, RD, P)는 사용자에게 노출하지 않음
- 한글 명칭(자극 추구, 위험 회피, 사회적 민감성, 인내력)만 사용

### AI 리포트 생성
- LLM: OpenAI API
- 출력 포맷: JSON
- 리포트 본문에 영문 약어 사용 금지 (프롬프트에 명시)
- 생성된 리포트는 DB에 캐싱

### 접근 권한 (유/무료)
- 무료: 히어로 영역, 기본 기질 분류, 레이더 차트, 관계 카드(이름+점수)
- 유료: AI 심층 분석, 양육 가이드, 실전 대사, 기질 차이 분석 등
- 유료 미결제 시: 블러+그레이스케일 처리 또는 생성 버튼 표시

---

## 1. 아이 진단 탭

| 순서 | 섹션 | 목적 | 접근 권한 |
|------|------|------|-----------|
| 1 | 히어로 영역 | 기질 유형 이미지, 유형명, 키워드, 한 줄 설명 | 무료 |
| 2 | 기질 점수 개요 | 4대 기질 점수 시각화 + AI 종합 분석 | 유료 |
| 3 | 아이나의 한마디 | 아이 기질 유형의 핵심 특성 해설 | 유료 |
| 4 | 기질 심층 분석 | 각 차원별 점수 기반 상세 해석 | 유료 |
| 5 | 아이의 숨겨진 속마음 | 오해하기 쉬운 행동 뒤의 진짜 마음 통역 | 유료 |
| 6 | 양육 가이드 | 상황별 구체적 양육 팁 | 유료 |
| 7 | 마법의 한마디 | 바로 따라 말할 수 있는 실전 대사 | 유료 |

### 세부 규칙

- **기질 점수 개요**: 점수는 0~100 범위, 프로그레스 바로 시각화
- **아이나의 한마디**: 아이의 기질적 특징 중심 (양육자 위로 X), 3문장 이상
- **기질 심층 분석**: 각 차원별 2~3문장, 일상 행동 예시 포함. `dimensions` 필드 없으면 미표시
- **아이의 숨겨진 속마음**: 2단락 이상, "겉으로 보이는 행동 vs 실제 마음" 구조
- **양육 가이드**: 최소 2개 상황, 각 팁에 기질적 근거 포함
- **마법의 한마디**: 3개 상황(응원/진정/칭찬), 자연스러운 구어체

## 2. 양육자 분석 탭

| 순서 | 섹션 | 목적 | 접근 권한 |
|------|------|------|-----------|
| 1 | 히어로 영역 | 양육자 기질 유형 이미지, 유형명, 키워드 | 무료 |
| 2 | 나의 기질 분석 | 양육자 자신의 기질적 특성 이해 | 무료 |
| 3 | 나의 마법의 계절 | 가장 빛나는 순간, 에너지 넘치는 상태 | 무료 |
| 4 | 마음의 가뭄 | 에너지 고갈 신호와 취약 상황 | 무료 |
| 5 | AI 심층 분석 | 타이틀 카드 + 심층 분석 + 솔루션 + 편지 | 일부 무료 |

### 세부 규칙

- **AI 심층 분석**: 타이틀 카드는 무료 공개, 나머지는 유료 (블러+그레이스케일 처리)
- **심층 분석 섹션**: nature(타고난 마음의 결) / shining(빛나는 순간) / vulnerability(취약점) 3개
- **마음 영양제**: 퍼스널 솔루션 3개 (이름 + 행동 지침 + 기질적 근거)
- **아이나의 편지**: 문학적이고 다정한 2인칭 톤

## 3. 기질 맞춤 양육 탭

| 순서 | 섹션 | 목적 | 접근 권한 |
|------|------|------|-----------|
| 1 | 히어로 영역 | AI 생성 관계 이름(harmonyTitle) + 한 줄 정의(oneLiner). 로딩 중 스켈레톤 표시 | 무료 |
| 2 | 기질 비교 레이더 차트 | 4대 기질 점수 시각 비교. 로딩 중 랜덤 애니메이션 | 무료 |
| 3 | 관계 카드 | 관계 이름 + 궁합 점수(%) + 한 줄 정의 | 무료 |
| 4 | 핵심 기질 차이 | 가장 큰 점수 차이 차원 분석 + 관점 전환 | 유료 |
| 4-2 | 마음이 잘 맞는 부분 | 점수 차이 15 이내인 차원의 공통점 + 강점. 해당 차원 없으면 미표시 | 유료 |
| 5 | 양육 원칙 | DO/DON'T 포함 양육 원칙 3개 | 유료 |
| 6 | 이럴 때 이렇게 | 상황별 대처법 + 실전 대사 3개 | 유료 |
| 7 | 양육 스타일 진단 | 현재 스타일 적합성 + 조절 포인트 | 유료 |
| 8 | 오늘의 한 마디 | 냉장고에 붙여둘 핵심 메시지 | 유료 |

### 세부 규칙

- **히어로 영역**: 설문 미완료 시 기질 유형 대신 스켈레톤 표시 (아이/양육자 탭도 동일)
- **레이더 차트**: 로딩 중 800ms 간격 랜덤 값 애니메이션, 완료 후 실제 점수로 전환
- **양육 원칙**: 범용적 육아 조언 금지, 이 기질 조합에 특화. 정확히 3개
- **이럴 때 이렇게**: 상황 + 아이 속마음 + 빠지기 쉬운 반응 + 권장 반응 + 실전 대사. 정확히 3개
- **실전 대사**: 바로 따라 말할 수 있는 자연스러운 구어체. 따옴표 없이 생성 (코드에서 이중 따옴표 방지 처리)
- **오늘의 한 마디**: 30자 이내
- **다시 분석하기**: 버튼 클릭 시 스크롤 최상단 이동 후 로딩 시작
