# 30-S05. 상담 및 미션 기록 화면 (Log & History)

## 1. 화면 개요
*   **화면 ID**: `SCR-LOG-001`
*   **경로**: `/record`
*   **설명**: 과거 상담 내역과 처방받은 미션 이력을 확인하는 대시보드.

## 2. UI 구성 요소
### 2.1 상단 필터 및 탭
*   **상급 탭**: [상담 기록] | [실천 미션]
*   **아이 선택**: 다자녀인 경우 특정 아이의 기록만 필터링 (현재는 단일 아이 중심).

### 2.2 상담 기록 카드 (History Card)
*   **아이콘**: 고민 카테고리별 심볼 이미지.
*   **텍스트**: 상담 날짜 (예: 2024.02.22), 카테고리 타이틀, 상황 설명 요약.
*   **버튼**: "처방전 다시보기"

### 2.3 미션 실천 로그 (Mission Log)
*   **상태**: 미실천(Default), 실천 완료(Check).
*   **내용**: "오늘의 마법 문장" 또는 "데일리 액션 아이템" 텍스트.
*   **완료 축하**: 체크 시 미세한 햅틱 반응 및 완료 애니메이션.

## 3. 디자인 가이드 (UX)
*   **톤앤매너**: 따뜻하고 차분한 일기장 느낌 (Beige & Wood 톤).
*   **Empty State**: 기록이 없을 경우 "아직 통역된 마음이 없어요. 마음 통역소에서 아이의 신호를 기록해보세요." 문구와 함께 상담 유도 버튼 노출.
*   **스크롤**: 무한 스크롤 또는 월별 세그먼트 구분.

## 4. 인터랙션
*   카드 클릭 시 `/consult/result/[id]` 또는 상세 모달 레이어로 이동.
*   미션 체크 박스 토글 시 데이터베이스 즉시 반영.
