# 단계 2: 990원 단건 액션 아이템(결제 및 마음 처방) 실행 계획

이 계획은 사용자가 소액 결제(990원)를 통해 즉각적인 육아 힌트와 위로를 얻는 '경험(Entry)' 단계를 구현하는 것을 목표로 합니다.

## 1. 개요
*   **목적**: 결제 허들을 낮추고 앱 설치 명분을 만드는 "마법의 열쇠" 기능 구현.
*   **주요 범위**: 
    1.  결제 페이지(`payment`) 가격 및 혜택 안내 고도화.
    2.  오늘의 마음 처방(Prescription) 데이터 및 전용 UI 구현.
    3.  기질별 '아이 신호 통역' 및 '마법의 한마디' 콘텐츠 연동.

## 2. 세부 작업 내역

### 2.1 결제 페이지(`payment`) 개편
*   [ ] **가격 업데이트**: $1 -> 990원 (한정 기간 특가)으로 변경.
*   [ ] **혜택 강조**: "오늘의 마음 처방", "아이 신호 통역", "나의 기질 카드" 등 990원으로 얻는 핵심 가치 시각화.
*   [ ] **신뢰 요소 추가**: 보안 결제 아이콘, 전문가 보증 문구 등 추가.

### 2.2 마음 처방(Prescription) 엔진 구축
*   [ ] **콘텐츠 라이브러리 제작**: `TemperamentClassifier`의 결과 타입별로 처방전 데이터 구축.
    *   `interpretation`: 아이의 신호 통역 (비유적 표현)
    *   `magicWord`: 부모가 바로 사용할 수 있는 문장 가이드
    *   `chemistryTheme`: 나의 양육 시너지 일러스트 테마 정보
*   [ ] **데이터 연동**: `useAppStore`에 결제 여부(`isPaid`)에 따라 리포트 페이지에서 처방전 섹션을 활성화.

### 2.3 리포트 페이지 내 처방전 전용 UI (`HeartPrescription`)
*   [ ] **상단 배치**: 결제 완료 후 리포트 진입 시 최상단에 "오늘의 마음 처방" 카드 노출.
*   [ ] **비주얼 카드**: 내 마음 기질과 아이 마음이 어우러진 감성적인 카드 디자인 (Glassmorphism 적용).
*   [ ] **애니메이션**: 처방전 카드가 나타날 때 부드러운 페이드인 및 위로 솟아오르는 효과 적용.

## 3. 기술적 변경 사항
*   `app/src/app/payment/page.tsx`: 990원 전용 결제 UI로 수정.
*   `app/src/lib/PrescriptionData.ts`: (신규) 기질별 처방 콘텐츠 라이브러리.
*   `app/src/app/report/page.tsx`: 결제 여부에 따른 처방전 섹션 추가 및 UI 개선.

## 4. 기대 효과
*   심리적 결제 장벽 완화 (커피 한 잔보다 저렴한 가격).
*   분석 데이터의 즉각적인 실천 도구 제공으로 사용자 만족도 제고.
*   브랜드 정체성(마음 기질, 처방전)이 담긴 비주얼 공유 유도.
