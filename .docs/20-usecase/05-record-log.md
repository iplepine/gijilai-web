# 20-U05. 상담 및 처방 기록 (Log)

## 1. 개요
*   **목적**: 사용자가 과거에 '마음 통역소'를 통해 상담받은 내용과 처방받은 '데일리 액션 아이템'을 한눈에 확인하고 관리함.
*   **사용자 가치**: 과거 아이의 행동 패턴을 돌아보고, 실천 중인 솔루션의 이력을 관리하며 양육 성찰의 도구로 활용.

## 2. 주요 기능
### 2.1 상담 내역 리스트 (History List)
*   과거 모든 상담 내역을 최신순으로 나열.
*   각 내역에는 상담 날짜, 고민 카테고리, 아이의 속마음 요약 노출.
*   클릭 시 상세 '마음 처방전' 화면으로 이동.

### 2.2 데일리 미션 아카이브 (Action Item Log)
*   처방받은 데일리 액션 아이템들의 목록 및 실천 여부 관리.
*   '미션 완료' 체크 기능을 통해 실천 동기 부여.

## 3. 사용자 플로우
1. 사용자가 하단 네비게이션 바에서 **'기록'** 탭 클릭.
2. **[상담 기록]**이 기본 탭으로 노출되며 과거 이력 리스팅.
3. 특정 상담 내역 클릭 시 해당 시점의 **[마음 처방전]** 상세 확인.
4. (선택) 상단 탭 전환을 통해 **[미션 목록]**만 따로 모아보기.

## 4. 정책
*   **데이터 보존**: 사용자가 계정을 삭제하지 않는 한 무기한 보존.
*   **정렬**: 최신 데이터가 상단에 노출 (Reverse Chronological Order).
*   **미션 연동**: 마음 통역소에서 '미션으로 등록'한 항목만 기록 탭의 미션 목록에 노출.
