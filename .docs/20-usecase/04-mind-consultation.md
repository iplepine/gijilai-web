# 마음 통역소 (육아 고민 상담 및 액션 관리)

## 1. 개요 (Overview)

**'마음 통역소'**는 부모가 겪고 있는 현재의 육아 고민을 입력하면, 아이와 양육자의 '마음 기질' 데이터를 기반으로 원인을 분석하고 즉각적으로 실천 가능한 맞춤형 처방을 내려주는 초개인화 상담 기능입니다.

*   **핵심 철학**: 모호한 위로 대신 정확한 기질적 원인 규명
*   **지향점**: "검사받는 느낌"이 아닌 "이야기 들어주는 전문가"와의 대화
*   **결과물**: 단발성 텍스트 조언이 아닌, 홈 화면과 연동되어 지속 실천 가능한 **'액션 아이템(Action Item)'** 구축

---

## 2. 사용자 시나리오 기반 플로우 (Natural User Flow)

### Scenario: "아침마다 옷 안 입겠다고 버티는 아이와의 전쟁"

#### Step 1. 감정 털어놓기 (Venting & Empathy)
- **부모의 행동**: "아침에 옷 안 입겠다고 30분 동안 울고불고... 결국 소리 질렀어요. 너무 힘드네요." (자유 형식 입력)
- **아이나의 반응**: 즉각적인 공감과 기질적 힌트 제공
  - *"정말 고생 많으셨어요. 아침 시간은 1분 1초가 급한데 아이가 협조해주지 않으면 누구라도 화가 날 수밖에 없어요. [아이이름]는 자기 주관이 뚜렷한 탐험가형 기질이라, 본인의 선택권이 존중되지 않는다고 느끼면 더 강하게 저항하는 특성이 있어요."*

#### Step 2. 맥락 파악 (Contextual Interaction)
- **아이나의 질문**: 딱딱한 문진표 대신 대화 흐름 속에서 상황의 맥락을 묻습니다.
  - *"혹시 그때 아이가 특정 옷의 촉감을 싫어했나요? 아니면 단순히 하던 놀이를 멈추기 싫어서였을까요?"* (선택지 혹은 단답형)
- **부모의 답변**: "그냥 자기가 고른 옷을 입겠다고 고집을 부렸어요."

#### Step 3. 실시간 마음 통역 (Immediate Insight)
- **아이나의 통역**: "지금 [아이이름]의 언어를 통역해 드릴게요."
  - *"아이의 속마음: '엄마, 나도 내 몸의 주인이고 싶어요! 내가 입을 옷은 내가 결정해서 나의 유능함을 확인하고 싶단 말이에요!'"*

#### Step 4. 맞춤형 마법의 솔루션 (Prescription)
- **마법의 한마디**: "내일 아침에는 이렇게 말씀해 보세요."
  - *"**[아이이름]야, 오늘 멋진 주인공이 될 준비 됐니? 이 파란 티셔츠랑 노란 티셔츠 중에 어떤 걸 입고 출동할까? 네가 직접 골라줄래?**"*
- **데일리 액션**: "아이에게 매일 2가지 선택지 중 하나를 고르게 하기" (미션 등록 유도)

## 3. 기술적 상담 프로세스 (Technical Flow)

### Phase 1: Dynamic Diagnostic (1차/2차 문진)
사용자가 입력한 고민 텍스트를 분석하여, 기질적 가설을 검증하기 위한 질문을 최대 5개까지 생성합니다.
1. **Initial Context**: 장소, 시간, 부모의 즉각 반응.
2. **Hypothesis Testing**: 특정 기질적 특성(예: 감각 예민성, 자율성 욕구) 여부 확인.

### Phase 2: Integrated Analysis
[아이 기질(CBQ)] + [양육자 기질(ATQ)] + [상담 맥락]을 매칭하여 충돌 지점(Chemistry Gap)을 도출합니다.

### Phase 3: Action Item Generation
가장 시급하고 실천하기 쉬운 솔루션 1개를 추출하여 데일리 미션으로 변환합니다.

---

## 4. 프롬프트 설계 (Conversational Prompt)

### 프롬프트: 공감 기반 통역사 '아이나' 페르소나
```markdown
당신은 기질 역동 분석 전문가입니다. 아래의 원칙을 지켜 답변하세요.
1. **공감 우선**: 부모의 고충을 충분히 인정하고 감정을 어루만져 주세요.
2. **죄책감 해소**: 갈등의 원인을 '부모의 잘못'이 아닌 '서로 다른 기질의 충돌'로 설명하세요.
3. **전문적 통역**: 아이의 문제 행동을 기질적 욕구(NS, HA, RD, P)의 관점에서 번역하세요.
4. **구체적 솔루션**: 모호한 조언 대신 즉시 사용할 수 있는 스크립트를 제공하세요.
```

---

## 5. 액션 아이템 관리 및 데이터 활용

1.  **미션 등록**: 상담 결과 페이지 하단에서 `[내일의 미션으로 등록]` 클릭.
2.  **홈 대시보드**: 등록된 미션이 메인 화면에 카드 형태로 노출.
3.  **성과 확인**: 일주일간의 미션 수행 결과를 '마음 성장 리포트'에 반영.

---

## 6. 기대 효과
- **UX 만족도**: 검사받는 기분에서 상담받는 기분으로 전환.
- **실천력 강화**: 읽고 잊어버리는 리포트가 아닌, 매일의 행동 변화로 유도.
- **데이터 축적**: 부모의 구체적인 육아 에피소드 데이터가 쌓여 더 정교한 개인화 추천 가능.
