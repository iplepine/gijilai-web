# Project Temperament

아동 기질(CBQ) × 사주 융합 분석 서비스

## 프로젝트 개요

- **서비스**: 과학적 기질 검사와 전통 사주를 결합한 맞춤형 육아 가이드
- **타겟 사용자**: 3~7세 자녀를 둔 부모
- **수익 모델**: $1.00 리포트 결제

## 핵심 기능

1. **기질 설문** - CBQ-VSF(아동), ATQ-SV(부모) 수집
2. **사주 분석** - 만세력 API 연동, 오행/십신 매핑
3. **융합 리포트** - 기질×사주 통합 성향 도출
4. **맞춤 솔루션** - 놀이법, 대화 스크립트, 환경 구성 제안

## 화면 플로우

```
홈 → 접수 → 설문 → 결제 → 리포트(메인/상세) → 솔루션 → 공유
```

- Frontend: React/Next.js, Tailwind CSS, Chart.js
- Backend: Node.js or Python
- Payment: Stripe/카카오페이
- External API: 만세력 API

## 핵심 알고리즘

### 기질 검사
- **CBQ-VSF**: 아동용 36문항, 3요인 (외향성, 부정적 정서, 의도적 통제)
- **ATQ**: 부모용 77문항, 4요인
- **T-점수 변환**: `T = 50 + 10 × (X - μ) / σ`

### 적합도(GoF) 계산
- 부모-자녀 기질 불일치도 + 부모 조절력 보너스
- 4가지 유형: 조화형, 보완형, 에너지불일치, 정서민감형

### 통합 분석
- 기질(과학) × 사주(전통) 교차 분석
- 일관성 → 핵심 성향, 상충 → 잠재력/위험구간

## 개발 시 유의사항

- 역채점 문항: `8 - score` 변환 필요
- 결측치(0): 평균 계산 시 분모에서 제외
- 아이 나이(개월): 백엔드에서 자동 계산 → 설문/솔루션 가중치 적용
- 사주 부정적 표현(살, 흉 등): 현대적 심리 키워드로 필터링 필수
