# 프로젝트 문서

모든 프로젝트 문서는 [docs/INDEX.md](docs/INDEX.md)를 참고하세요.

# 코드 변경 시 문서 동기화

- **정책 변경**: 비즈니스 룰, 점수 로직, 결제, 설문, 네비게이션 등이 바뀌면 `docs/policies/` 해당 문서도 함께 업데이트
- **의사결정(ADR)**: 구조, 정책, 기술 선택 등 의사결정이 발생하면 `docs/ADR.md`에 기록
  - 형식: `## YYYY-MM-DD | 결정 제목` + 결정/이유/대안
  - 사소한 변경(오타, 스타일)은 제외. 왜 그렇게 했는지 나중에 궁금할 만한 것만 기록
